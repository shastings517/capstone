Tweeter visualizes realtime sentiment from tweets. Get started by entering a keyword in the input box and pressing the start button ().

Tweeter will start graphing sentiment(positive/negative) data parsed from tweets about your keyword in realtime. Tweet sentiment is represented on the y-axis and time on the x-axis. Tweets that are positive will appear above the x-axis while negative tweets will appear below it.

Click the circular "M" button above the graph to see where the tweets are coming from. See the color gradient at the top of the screen for scale; green represents positive, red represents negative, and yellow represents neutral sentiment. The darker the color the more positive/negative the sentiment.

Click the circular "D" to see more data about the tweet stream.

Click the reset button () to reset and search again.
